### ğŸ“– About
**HealthCalc** is a mobile application developed in React Native (Expo) designed to be your personal health assistant. The app focuses on accessibility and a clear User Experience (UX), featuring a modern **Glassmorphism** interface.

It calculates essential health metrics like BMI, daily water intake, and Basal Metabolic Rate (using the Harris-Benedict equation) to generate a personalized action plan for weight loss, maintenance, or muscle gain.

### âœ¨ Features
* **BMI Calculator:** Calculates Body Mass Index with color-coded diagnoses (Underweight, Healthy, Overweight, Obesity).
* **Hydration Tracker:** Estimates daily water needs based on body weight and exercise frequency.
* **TDEE Calculation:** Calculates Total Daily Energy Expenditure using the Harris-Benedict equation, considering:
    * Age, Gender, Height, Weight.
    * Job activity level (Office vs. Physical).
    * Exercise frequency.
* **Action Plan:** Automatically generates caloric goals and exercise tips based on the user's ideal weight.
* **Modern UI:**
    * Beautiful Linear Gradients.
    * Glassmorphism components (blurred transparencies).
    * **Dark/Light Mode** toggle.
* **History:** Saves previous calculations locally for quick reference.
